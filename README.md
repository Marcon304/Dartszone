Dieses Projekt entstand als vollständiges Einzelprojekt im Rahmen des Schulfachs Seminarfach mit dem Thema Design.
Ziel war die Entwicklung eines theoretisch verkaufsfähigen Produkts – von der Idee über das Konzept bis zur Umsetzung.
Mein Produkt war die Konzeption und Umsetzung einer Webshop-ähnlichen Website mit Fokus auf User Interface und Webdesign.

Der Schwerpunkt lag bewusst auf Struktur, Gestaltung und Nutzerführung.
Typische Shop-Funktionen wie der Produktfilter sind daher teilweise nur als visuelle Platzhalter umgesetzt.

## Features
- Eigenständig entwickeltes, modernes Layout
- Produktübersicht mit klarer Kategorienstruktur
- Simulierter Warenkorb und Checkout-Ablauf zur Darstellung typischer Shop-Prozesse

## Technologien
- HTML
- CSS
- JavaScript

## Projektumfang
- Eigenständige Planung und Umsetzung als Einzelprojekt
- Strukturierung der Inhalte und Seitenarchitektur
- Umsetzung der grundlegenden JavaScript-Logik für Navigation und Warenkorbsimulation

## Lernziele und Erkenntnisse
- Entwicklung eines konsistenten User Interfaces
- Trennung von Struktur (HTML), Design (CSS) und Funktionalität (JavaScript)
- Bedeutung von Nutzerführung und klaren Abläufen im Webdesign
- Selbstständiges Arbeiten von der Idee bis zum Ergebnis

## Hinweis
Dies ist kein produktiver Online-Shop, sondern eine Demo-Version für Präsentationszwecke.
Einige Produktbilder und Markennamen wurden mit vorheriger Genehmigung von dartshopper.de verwendet.
